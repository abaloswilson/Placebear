# PlaceBearAPI

A small static web demo. Open `index.html` in a browser to view the project.

## Contents
- `index.html` — main HTML page
- `script.js` — JavaScript for the demo
- `style.css` — styles for the page

## Usage
1. Open `index.html` directly in your browser, or serve the folder with a simple HTTP server:

   ```bash
   python -m http.server 8000
   ```

2. Visit `http://localhost:8000` and open `index.html`.

## Development
- Edit `index.html`, `script.js`, and `style.css` and refresh the page to see changes.

## Notes
- This is a static site — no build step required.
